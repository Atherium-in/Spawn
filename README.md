# Fruit Game: Merge and Smile — Legal Pages

This repository hosts the public **Privacy Policy** and **Terms of Service** for the game *Fruit Game: Merge and Smile*.

## Files
- `index.html` — simple landing page linking to both documents
- `privacy-policy.html` — HTML privacy policy (AdMob + Google Play Billing, no user data collection, non-refundable purchases)
- `terms-of-service.html` — HTML terms (includes Pixabay audio attribution)
- `privacy-policy.md` — Markdown version of the privacy policy
- `terms-of-service.md` — Markdown version of the terms

## GitHub Pages
1. Push this repo to GitHub.
2. In **Settings → Pages**, set **Branch: main** and **Folder: / (root)**.
3. Your site will be live at: `https://<your-username>.github.io/<your-repo>/`
